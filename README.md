介绍(Introduction)
----------
Orc manner是一个用于[Orc-Engine](https://github.com/baka397/Orc-Engine)策略包.  
Orc manner is a strategy package for Orc-Engine.

安装(Install)
----------
使用`npm`安装Orc  
Use `npm` to install Orc:
```
npm install orc-manner --save
```

测试(Test)
----------
```
npm test
npm run test-cov
```
